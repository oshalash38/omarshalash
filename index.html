Objectively Best Calendar

School project for CSC207 class. Worked in a team to develop a fully functional desktop calendar app using Java (Swing). Used what I learned from the course including but not limited to design principles, design patterns and app architecture guidelines dictated by the Clean Architecture book (good read). Worked on both the back-end as well as the front-end, and implemented features such as the event-related and memo-related features (creating/editing/displaying/deleting events and memos), messaging features and login features as well as overall bug fixes.

Link-Up

Project for HackTheValley IV and my first experience developing mobile apps. Used Java and Android Studio to develop a social networking app intended to arrange meet-ups in a simple manner. I was personally responsible for designing and implementing the front-end and figuring out the flow and logic of the application. Furthermore, I was exposed to new technologies such as Google Firebase.
